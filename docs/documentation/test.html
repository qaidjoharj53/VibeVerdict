Setting Up NLTK Environment Documentation¶

This documentation provides step-by-step instructions for setting up a Python environment, installing NLTK (Natural Language Toolkit), and running a sample program that involves NLTK for tokenization, POS tagging, and sentiment analysis.

Creating a Virtual Environment¶

  1. Open a terminal or command prompt.

  2. Navigate to the project directory (if not already there).

  3. Create a virtual environment using the following command:

    python -m venv venv

  4. Activate the virtual environment:

    • On Windows:

      venv\Scripts\activate

    • On macOS/Linux:

      source venv/bin/activate

Installing NLTK and Dependencies¶

  1. With the virtual environment activated, install NLTK and its components:

    pip install nltk

  2. Download NLTK data for tokenization, POS tagging, and sentiment analysis. In a Python script or interactive environment, run the following:

    import nltk

nltk.download('punkt')
nltk.download('averaged_perceptron_tagger')
nltk.download('vader_lexicon')

Running the Existing Test Script¶

  1. There is already a test script available for NLTK in the test folder. The file name is nltk_test.py. To run the test script, execute the following command in the terminal:

    python test/nltk_test.py
