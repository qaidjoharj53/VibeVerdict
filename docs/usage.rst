Using Your Flask App
====================

Installation
------------
To install your Flask app, run the following command:

.. code-block:: bash

   pip install myflaskapp

Usage
-----
1. Run the app:

   .. code-block:: bash

      python myflaskapp.py

2. Access the app in your web browser at http://localhost:5000.
